  The current ENC driver version is 2.2.1.

  - 2.2.1
    - Improvements
      - Release peripheral from reset if necessary in init function.

  - 2.2.0
    - New Features

  - 2.1.0
    - Improvements
      - Supported period measurement function.

  - 2.0.2

  - 2.0.1
    - Bug Fixes
      - Fixed violations of the MISRA C-2012 rules 10.1, 10.3, 10.4, 10.6,
        17.7.

  - 2.0.0
